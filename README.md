# r6wins
#### Video Demo: 
#### Description:
  The project gets user's input on his gameplay results - for example: operator used this round; win or lose in the end; number of kills; number of death;
  then it calculates the K/D ratio and winrate of every operator that user had played with. After the exit of terminal (Ctrl+C) 
  it deploys the statistics into a .json and .csv tables so it's easier to understand and visualise it.
